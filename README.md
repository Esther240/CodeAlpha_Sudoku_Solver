# CodeAlpha - Sudoku Solver (C++)

## Description (TASK 4)

A C++ program that solves a 9x9 Sudoku puzzle using a backtracking algorithm.

## Features

* Solves Sudoku using recursion
* Validates row, column, and 3x3 grid rules
* Displays original and solved puzzle

## How to Run

g++ sudoku.cpp -o sudoku
./sudoku
